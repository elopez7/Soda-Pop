The Code for THE 300

                        
                            function getValuesFromPage(){
                                let startValue = document.getElementById("startValue").value;
                                let endValue = document.getElementById("endValue").value;
                                let generatedNumbers = [];
                            
                                startValue = parseInt(startValue);
                                endValue = parseInt(endValue);
                            
                                if(Number.isInteger(startValue) && Number.isInteger(endValue)){
                                    generatedNumbers = generateNumbersFromStartValueToEndValues(startValue, endValue);
                                }
                                else{
                                    alert("You must enter integers")
                                }
                            
                                displayNumbers(generatedNumbers);
                            }
                            
                            function generateNumbersFromStartValueToEndValues(startValue, endValue){
                                let generatedNumbers = [];
                            
                                for(let index = startValue; index <= endValue; index++){
                                    generatedNumbers.push(index);
                                }
                            
                                return generatedNumbers;
                            }
                            
                            function displayNumbers(numbersToDisplay){
                                let templateRow = "";
                                for (let index = 0; index < numbersToDisplay.length; index++) {
                                    let number = numbersToDisplay[index];
                                    templateRow += markEvenNumbersBold(number);
                                }
                                document.getElementById("results").innerHTML = templateRow;
                            }
                            
                            //Note that HTML elements don't render correctly in Prism, see the source.
                            function markEvenNumbersBold(numberToMark){
                                let className = "even"
                                if(numberToMark % 2 == 0){
                                    return `${numberToMark}`;
                                }
                                className = "odd";
                                return `${numberToMark}`;
                            }

The Code is structured in four functions.

getValuesFromPage()

Takes the user input from the page and stores the variables in startValue and endValue then the input is converted to integer values by using parseInt(), if the input is correct then a call to generateNumbersFromStartValueToEndValues() will populate the generatedNumbers with the generated count. The array then will be passed to displayNumbers(), so that the generated values can be printed on the page.

generateNumbersFromStartValueToEndValues()

Will take the values of the startValue and endValue parameters and will use them in the for loop. A loop is used to generate the values between startValue and endValue while storing them in the generatedNumbers array.

displayNumbers()

Takes an array that contains the numbers to be displayed. A templateRow string variable is created. This string will hold the values to be displayed along with html classes that will define the ones to be shown in bold. A loop to iterate through the values stored in the numbersToDisplay array and markEvenNumbersBold() is called to decide which value will be printed in bold.

markEvenNumbersBold()

Will decide if the numberToMark parameter is even or odd. Depending on the result the html className variable will be concatenated with the value and returned as a string to be added to templateRow.
